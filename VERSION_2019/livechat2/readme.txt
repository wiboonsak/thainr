     โปรแกรมนี้ถูกพัฒนาขึ้นเพื่อศึกษาการทำงาน ของระบบ ที่เรียกว่า Ajax (Asynchronous JavaScript and XML) ที่เป็นเทคโนโลยีที่ถูกสร้างขึ้นเพื่อทำการรับส่งข้อมูล
ระหว่าง Server -> Client โดยไม่ต้องผ่านการ Submit หน้า Form แต่อย่างใดแต่ผ่านทาง XMLHttpRequest แทน ซึ่งเทคนิคนี้เราสามารถนำมาใช้งานการส่งข้อมูล
โดยไม่ต้องทำการ Reload หน้าใหม่แต่อย่างใด นี่จึงเป็นที่มาและหลักการ ของ โปรแกรม Chat Room ตัวนี้
(อ้างอิงมาจาก : http://www.unzeen.com)

การติดตั้ง :
1. นำไฟล์ .sql ที่อยู่ภายใน Folder -> SQL สร้าง Database ของคุณขึ้นมา โดยค่า Default เป็น ajaxchat (สามารถแก้ไขได้ในไฟล์ config.inc.php)
2. แก้ไขข้อมูลของคุณในไฟล์ที่ชื่อว่า /libs/config.inc.php จะเป็นข้อมูลเรื่อง Database เป็นส่วนใหญ่
3. ถ้าคุณใช้ MySQL เวอร์ชั่นต่ำกว่า 4.1.12 ให้แก้ไฟล์ /libs/mysql.php ในบรรทัด $this->query('set charset tis620'); ให้ comment ทิ้ง
4. ถ้า Host ที่คุณใช้เป็น Unix ต้อง CHMOD ไฟล์ /data/messages.html เป็น 777 ก่อน ไม่นั้น message จะไม่สามารถเขียนได้เลย

ไฟล์ที่นำมาใช้ร่วมกับโปรแกรมนี้
class ของ mysql function อันนี้ผมแกะออกมาจาก ipb ครับ
function ของ ajax ที่ชื่อว่า sajax อันนี้ผมโหลดมาจากไหนไม่รู้ แต่มันเป็นของฝรั่ง

(โปรแกรมนี้เป็นแค่ตัว Beta เท่านั้น ยังไม่สมบูรณ์ดีนัก)

เพิ่มเติมใน Version เต็ม
1. มีเสียงเวลาทำ Action ต่างๆ
2. ส่งข้อความเป็นรายบุคคลได้
3. สามารถแบ่งห้องได้ โดยข้อความจะถูกแยกกับเก็บ
4. จำกัดจำนวนคนเข้าในแต่ละห้องเพื่อไม่ให้เรียกไฟล์เดียวกันจำนวนมาก
5. ป้องกันการ Flush Message โดยกำหนดเวลาได้
6. สามารถเลือกสีสำหรับการพิมพ์ได้
7. สามารถเลือกเพศเพื่อเข้าห้องได้
8. โหมดการพิมพ์เปลี่ยนเป็น Design Mode ในลักษณธ Word
9. มีระบบ Admin สำหรับควบคุม ห้องต่างได้
